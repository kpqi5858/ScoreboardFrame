# ScoreboardFrame
Awesome scoreboard plugin for Minecraft 1.5.2 or later

## Features
 - Pretty Customizable.
 - Like gif files, All scoreboard is based "Frame".
 - Using PlaceholderAPI to get Player's a lot of information.
